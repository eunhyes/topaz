# 🍀 TOPAZ 실버타운 그룹웨어 및 고객 페이지

## 💡 프로젝트 소개
고령화 인구가 증가함에 따라 다양한 형태의 실버타운이 등장할 것으로 예상하며 실버타운 그룹웨어 및 고객 페이지 구현
- 배포 URL : <http://13.209.193.10/topaz/login>
- 개발 기간 : 2024.06.24 ~ 2024.08.02
- 프로젝트 시연 영상 : [전체 시연 영상 보기](https://youtu.be/wenoEmmYZUw)

**개발 환경**
<br>
OS : Mac OS sonoma <br>
Back-end : Java <br> 
Front-end : HTML5, CSS3, BootStrap <br>
Library : JQuery, JSTL, Lombok, MyBatis, Maven <br>
Database : MariaDB <br>
WAS : Apache Tomcat 10 <br>
IDE : STS4 <br>
Cloud Service : Amazon Web Services <br>
버전관리 : GitHub <br>
<br>

## 💫 담당 기능
### 로그인
- 세션을 활용한 로그인 기능 구현
- 필터를 활용하여 접근 권한 부여 및 제한

### 결재 리스트
- REST API와 AJAX를 활용한 리스트 구현
- AJAX를 활용한 비동기적 페이징, 검색 기능 구현

### 봉사 일정 관리
- Full Calendar를 이용한 일정 관리 기능 구현

### 봉사 신청 관리
- REST API와 AJAX를 활용한 리스트 구현
- AJAX를 활용한 비동기적 페이징, 검색 기능 구현

### 고객페이지
- 회원가입
  - 구글 SMTP와 Spring Mail 라이브러리를 활용한 이메일 인증 기능 구현
  - 다음 카카오 주소 API 활용

- 로그인 
  - 세션을 활용한 로그인 기능 구현
  - 필터를 활용하여 접근 권한 부여 및 제한

- 비밀번호 찾기 
  - 임시 비밀번호를 발급 후 이를 통해 고객이 로그인할 수 있도록 기능 구현

- 봉사 신청
  - Full Calendar를 이용하여 그룹웨어에서 등록한 봉사 일정을 볼 수 있도록 구현
  - 봉사 신청 인원이 정해진 수용 인원을 초과할 경우 신청이 불가하도록 제어 기능 구현

- 마이페이지
  - 비밀번호 수정 
  - 내 정보 수정 및 탈퇴
  - AJAX를 활용하여 내가 신청한 봉사 리스트를 볼 수 있도록 구현
